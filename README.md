# Todo API

A simple **Node.js & Express** backend API for managing todo items (Create, Read, Update, Delete).

## Features

- Add new todo items
- View all todos
- Update todo items
- Delete todo items

## Getting Started

### Prerequisites

- Node.js (v16+ recommended)
- npm
- MongoDB (local or cloud, e.g., MongoDB Atlas)
